## Introduction

Tooling is a CLI tool to help you fast develop any JavaScript app. And it's very modular, using plugins to achieve it's goals.

## Installation

```bash
# node >= 4
$ npm install -g tooling
```

## Usage

For A `webpack` fan:

```bash
$ cd my-project
$ tooling app.js --use webpack --use babel --use postcss
# thank god, it's done! simple as that.
```
